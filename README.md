# Guitar Practice Generator

A focused practice tool that generates random guitar exercises from curated sets.

## Features

- **4 preset modes**: Beginner Essentials → Scales & Triads → Harmony & Extensions → Advanced Integration
- **21 exercises** across fretboard mapping, scales, triads, seventh chords, voice leading, arpeggios, and progressions
- **Smart filtering**: Key selection, tonality (Major/Minor), modal options
- **Mobile optimized** with swipe navigation
- **No installation**: Single HTML file, works offline

## Usage

Visit [https://cornee.github.io/practice/](https://cornee.github.io/practice/) or open `index.html` locally.

1. Select a preset or customize your exercise pool
2. Choose key/tonality filters (optional)
3. Click **Generate** for a random exercise
4. Blue dot indicates pending filter changes

# Guitar Practice Generator

A focused practice tool that generates random guitar exercises from curated sets.

## Features

- **Preset modes**: Beginner Essentials → Scales & Triads → Harmony & Extensions → Advanced Integration
- **Tons of exercises** across fretboard mapping, scales, triads, seventh chords, voice leading, arpeggios, and progressions
- **Smart filtering**: Key selection, tonality (Major/Minor), modal options
- **Mobile optimized** with swipe navigation
- **No installation**: Single HTML file, works offline
